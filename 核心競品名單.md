# 核心競品名單｜Joyce 高階經理人領導力課程

## 結論

本名單是「Joyce 應該深入比較的競品」，不是完整市場資料庫。

依 Joyce 最新判斷：

| 處理方式 | 對象 | 原因 |
|---|---|---|
| 排除第一圈比較 | 哈佛企管 | 偏國際版權課、標準化主管訓練，和 Joyce 想做的高階經營/個案/社群型產品不同 |
| 排除第一圈比較 | 經理人商學院 | 偏大眾經理人與單日技能課，價格帶與客群層級較低 |
| 納入第一圈 | HBR個案共學會 | 同屬高階領導者、企業主、二代與決策型學習，且和領導影響力學院同屬遠見天下文化產品線 |
| 納入第二圈 | 天下學習 / 天下創新學院 | 是企業學習與內容平台競品，但不一定是線下高階經理人 cohort 的直接競品 |

## 第一圈：直接競品

這些課程會搶同一批高階學員、同一筆學習預算，應優先深入拆解。

| 競品 | 類型 | 為什麼要比 |
|---|---|---|
| HBR個案共學會 | 個案教學 / 高階共學 | 高階主管、企業主、二代；用個案討論提升決策力，是 Joyce 最需要研究的產品型態 |
| 領導影響力學院｜陳百州經營決策實戰班 | 經營決策 / 財務策略 / 成長策略 | 企業主與核心團隊一起上課，接近顧問式高階工作坊 |
| 領導影響力學院｜蘇欽豐 AI組織效能實戰課 | AI組織效能 / 高階主管AI導入 | 從工具課升級為組織流程與管理系統，和 Joyce 的 AI 領導力方向高度相關 |
| 領導影響力學院｜陶韻智給企業主管的AI實戰課 | AI決策 / 主管實作 | 用知名企業高階主管講師包裝 AI 實戰，值得比較招生語言 |
| 領導影響力學院｜李聖珉問題解決與批判思考 | 決策力 / 結構化思考 | 麥肯錫問題解決框架與企業實務結合，是高階主管決策課直接競品 |
| 商周百億CEO班 | CEO高價學程 | 高單價、審核制、企業主 cohort，是高階課程天花板級競品 |
| 商周CEO50失敗學 | 企業主個案學習 | 用失敗案例做企業主學習，和 HBR 個案教學形成對照 |
| 商周雙軌轉型組織領袖班 | 組織轉型 / 高階領導 | 企業主與中高階主管轉型課，適合比較產品結構與價格帶 |
| 商周EVP人才戰略課 | 人才策略 / CEO與HR共學 | 把人才留任、雇主品牌提升為 CEO 議題 |
| 企業經理協進會｜高階主管研習班 | 高階主管研習 / 人脈型學習 | 額滿訊號明確，具公信力與高階社群意味 |
| DDI｜高階評鑑與接班人才盤點 | 顧問式人才發展 | 若 Joyce 課程要服務企業接班與高潛發展，DDI 是方法論型競品 |
| 台大EMBA / 政大EMBA / 成大EMBA / 東海EMBA企業二代組 | 學位型替代品 | 不一定課程型競品，但會搶高階學員的時間、預算與人脈需求 |

## 第二圈：預算替代與生態系競品

這些不一定直接搶同一門課，但會搶企業學習預算、內容信任與企業內訓入口。

| 競品 | 判斷 | 追蹤重點 |
|---|---|---|
| 天下學習 / 天下創新學院 | 第二圈競品 | 企業學習平台、線上課程、實體工作坊、企業優勢診斷 |
| 天下學習｜EMBA全球大師視野 | 第二圈偏直接 | 若 Joyce 要賣「高階視野與大師課」，這條產品線會競爭 |
| 天下學習｜AI Agent落地實戰 | 第二圈偏直接 | 若 Joyce 做 AI 轉型與高階主管 AI 工作坊，需要比較 |
| 天下學習｜管理職能 / 關鍵領導力培訓地圖 | 第二圈 | 若 Joyce 做企業內訓或主管梯隊培訓，需要比較 |

## 天下學習是否算競爭者？

答案：算，但不是第一圈直接競品；應列為第二圈競品。

判斷如下：

| 情境 | 是否競爭 | 原因 |
|---|---|---|
| Joyce 做線下高階經理人 cohort / 個案討論 / 企業主社群 | 間接競爭 | 天下學習偏平台化、內容型、企業學習包，不是同一種高接觸產品 |
| Joyce 做企業內訓、AI學習方案、主管梯隊課程 | 直接競爭 | 天下學習有企業客製方案、實體工作坊、平台試用與管理職能課程 |
| Joyce 做線上內容訂閱或大師課 | 直接競爭 | 天下學習與天下創新學院在內容、品牌與學習平台上有優勢 |

## Joyce 應該怎麼比

建議後續不要再把所有課程放在同一張表比較，而是分三張表：

| 比較表 | 目的 |
|---|---|
| 第一圈直接競品表 | 比價格、課程結構、師資、對象、招生語言 |
| 第二圈預算替代表 | 看企業學習預算會被哪些平台或學程吸走 |
| 同集團產品線表 | 比 HBR個案共學會、領導影響力學院、HBR企業學習服務之間的定位差異 |
