## 研究範圍

本版聚焦台灣市場近期可查的線下/實體高階經理人、主管領導力、CEO、EMBA、顧問公司與媒體型管理教育課程。  
「課程」包含公開班、工作坊、論壇、EMBA學程、長學程中的明確模組。

## 欄位判讀

| 欄位 | 說明 |
|---|---|
| 資料信心 A | 官方頁資訊清楚，價格/日期/定位至少多數可查 |
| 資料信心 B | 官方頁可查，但部分欄位如師資或價格未完整公開 |
| 資料信心 C | 主要來自活動平台搜尋卡片，需二次確認 |
| 熱賣程度 | 以額滿、完售、好評加碼、平台瀏覽/收藏、多梯次開課等公開訊號判斷 |
| 口碑 | 只採公開頁面可見資訊，不推測私下評價 |

## 管理意義摘要

1. 高階課程市場已分成三層：數千元媒體型單日課、數萬至十萬級工作坊、百萬級CEO長學程。
2. AI不再只是工具課，而是被包裝成決策、流程、組織效能與高階主管轉型能力。
3. 人才策略、EVP、接班與高潛人才評鑑正在與CEO課程結合，顯示人才議題已升級為經營議題。
4. EMBA仍以學位、人脈與校友網絡為核心競爭力，與非學位高價CEO班形成替代與互補關係。
5. 顧問公司偏向方法論與企業內訓，媒體型學院偏向名師、案例、同儕社群與招生語言。
