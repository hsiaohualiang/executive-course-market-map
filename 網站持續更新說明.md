# 網站持續更新說明

更新日：2026-06-22

## 結論

這個網站不應只是一份靜態報告，而應作為「高階經理人課程市場情報站」持續更新。

目前網站採三層資料：

| 資料層 | 檔案 | 用途 |
|---|---|---|
| 市場基礎資料 | `site/data.js` | 放大量課程卡片、類別、主辦、價格、師資、來源 |
| 競品更新資料 | `site/market-updates.js` | 放第一圈競品、HBR補充資料、招生頁語言分析、產品結構拆解 |
| 客群與口碑規則 | `site/audience-rules.js` | 放目標客群、購買情境、適用企業、口碑可信度分級與補強來源 |
| 推薦開課資料 | `site/recommendations.js` | 放首頁六門建議遠見領導力學院優先開的課、新興類別候選、老師與招生文案 |

這樣做的好處是：大量爬取資料可以定期重建，但第一圈競品的判斷、詢問後補到的價格、名額、師資、招生話術，可以快速手動補進網站，不會被重建覆蓋。

## 建議更新頻率

| 競品層級 | 更新頻率 | 追蹤重點 |
|---|---|---|
| 第一圈直接競品 | 每月一次；開課前後每兩週一次 | 價格、名額、完售、加開、師資、招生頁話術 |
| 第二圈預算替代 | 每季一次 | 新課程、新企業方案、新平台產品 |
| EMBA學位型替代 | 每學年招生季更新 | 學費、名額、招生組別、課表與說明會 |
| C級長尾課程 | 半年一次 | 只看趨勢，不逐筆深查 |
| 新興類別與建議開課 | 每週一次 | 新老師、新課程、新類別、六門建議開課是否需要調整 |

## 每次更新要看什麼

| 欄位 | 判斷意義 |
|---|---|
| 日期與場次 | 看是否持續開課、是否加開 |
| 價格 | 看價格帶是否上移、是否改為諮詢制 |
| 名額 | 看稀缺性與班級定位 |
| 是否額滿/完售 | 判斷熱度 |
| 老師/顧問 | 看權威背書是否升級 |
| 學員對象 | 看是否從主管課升級成 CEO/企業主課 |
| 目標客群 | 分辨官方明載、類別推論、研究推論，不混為一談 |
| 招生頁語言 | 看競品如何描述痛點與承諾 |
| 產品結構 | 看是否加入診斷、作業、教練、陪跑、社群 |
| 用戶回饋可信度 | 分辨第三方回饋、官方具名見證、弱訊號與無資料 |
| 新興類別 | 不限制在20類；只要市場出現新需求，就先列為候選類別 |
| 建議開課 | 每週檢查首頁六門課是否仍是遠見領導力學院最該開的課 |

## 實際維護方式

1. 官方頁有新資料時，先更新 `site/market-updates.js`。
2. 若是新增第一圈競品，把課程放進 `extraCourses`。
3. 若是既有課程的新分析，更新 `insightRules`。
4. 若是補目標客群或口碑可信度，更新 `site/audience-rules.js`。
5. 若是發現新老師、新課程、新類別或新的熱門需求，更新 `site/recommendations.js`。

## 新類別與建議開課原則

類別不限制在 20 個。每週更新時，若市場出現新的高階主管學習需求，先放入 `site/recommendations.js` 的 `emergingCategories`。

首頁固定顯示六門「建議遠見領導力學院優先開課」，每門課需包含：

1. 課程類別。
2. 課程名稱。
3. 建議老師或師資組合。
4. 目標客群。
5. 市場訊號。
6. 課程產品結構。
7. 建議價格定位。
8. 招生頁文案重寫。
9. 參考來源。

選課判斷以六項為主：

| 判斷項目 | 說明 |
|---|---|
| 最新熱門訊號 | 是否有近期開課、加開、額滿、社群討論或競品投入 |
| 遠見品牌適配 | 是否適合遠見/領導影響力學院的高階主管定位 |
| 高階買單力 | 是否有明確企業主、CEO、二代、HR高層或高階主管買方 |
| 老師可信度 | 是否能找到有公信力且適合此題目的師資 |
| 差異化 | 是否避開商周、HBR、EMBA、DDI已佔據的心智 |
| 文案可賣性 | 是否能寫成清楚痛點、承諾與行動召喚 |

## 口碑資料可信度分級

| 等級 | 定義 | 處理方式 |
|---|---|---|
| A | 有第三方可驗證資料，例如具名學員公開貼文、獨立媒體訪談、公司職稱可交叉查核 | 可作為高信心口碑 |
| B | 有官方具名見證、公開學員數、額滿/加開/多梯次等訊號 | 可用，但需標示來自主辦方 |
| C | 只有官方品牌敘述、平台瀏覽收藏或匿名好評 | 只能當弱訊號 |
| D | 沒有可辨識回饋 | 標示待補，不推論 |
 
更具公信力的資料來源，優先順序如下：

1. 具名學員公開貼文。
2. 獨立媒體訪談。
3. 公司官網或新聞稿驗證學員職稱。
4. 主辦方提供的 NPS、續報率、完課率、職級與產業分布。
5. 3 到 5 位歷屆學員的一手訪談。

## 維護原則

1. 不推測未公開價格；只標示「未公開/需洽詢」。
2. 熱賣程度只用可觀測訊號：額滿、完售、加開、多梯次、官方學員數、公開見證。
3. 第一圈競品要分析「怎麼賣」，不只分析「賣什麼」。
4. 每次更新都要留下更新日與來源頁。
